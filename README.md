# 👣 Chronic Life Challenge 👣

만성질환(고혈압·당뇨) 생활습관 챌린지 웹 서비스 프로젝트

- 프로젝트 기간: 26.05.06 ~ 26.06.19 (38일)
- 팀원: 김나현, 김서영, 이로건, 주수빈

## 프로젝트 목표

공개 의료/건강 데이터셋을 활용해 사용자의 만성질환 발병 가능성을 예측하고, 
건강 지표 변화 추이를 대시보드로 보여주며, 
생활습관 챌린지를 통해 사용자가 꾸준히 개선 행동을 실천하도록 돕는 웹 서비스입니다.

## 필수 구현 기능

| 구분 | 주요 기능 | 설명 | 우선순위 |
|---|---|---|---|
| 회원관리 | 이메일 회원가입/로그인, 로그아웃, 내 정보 조회 | 사용자는 이메일 기반으로 가입/로그인하고 본인 정보를 조회할 수 있습니다. | 높음 |
| 건강 데이터 | 혈압 기록, 혈당 기록, 행동 데이터 기록 | 사용자는 혈압, 혈당, 걸음수, 수분 섭취량, 운동 시간, 수면 시간 등 건강·생활습관 데이터를 입력할 수 있습니다. | 높음 |
| 건강 점수 | 일일 점수 계산, 오늘 점수 조회, 기간별 이력 조회 | 입력된 건강 데이터를 기반으로 일일 복합 건강 점수를 계산하고, 기간별 점수 변화를 확인할 수 있습니다. | 높음 |
| 예측 모델 | 위험도 예측 요청, 예측 결과 조회 | 공개 데이터셋 기반 ML 모델을 활용해 사용자 건강 데이터 기반 만성질환 위험도를 예측합니다. | 높음 |
| 대시보드 | 종합 대시보드, 건강 수치 차트, 건강 점수 추세 차트 | 건강 지표, 건강 점수, 예측 위험도, 챌린지 현황을 차트와 요약 정보로 제공합니다. | 높음 |
| 챌린지 | 챌린지 목록 조회, 챌린지 참여, 챌린지 달성 처리 | 사용자는 생활습관 개선 챌린지에 참여하고, 행동 데이터 입력을 통해 달성 여부를 기록할 수 있습니다. | 높음 |
